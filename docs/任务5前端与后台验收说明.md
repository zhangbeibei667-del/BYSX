## 5. 运行说明

```bash
python -m uvicorn backend.main:app --host 127.0.0.1 --port 8000
```

生产构建已经生成在 `frontend/dist`，FastAPI 会直接托管该目录。管理员首次启动密码保存在本机忽略提交的 `data/.admin_initial_password`，也可通过 `ADMIN_PASSWORD` 环境变量配置。

构建仅保留 Sass 旧 `@import` 和主包体积提示，两者不影响功能与运行；后续如需做性能专项，可通过路由懒加载和 Sass `@use` 迁移继续优化。
